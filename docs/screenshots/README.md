# Скриншоты тестирования (TC-01—TC-12)

Данная папка содержит скриншоты, подтверждающие результаты ручного функционального тестирования веб‑приложения взаимодействия сотрудников колледжа с родителями.

## Связанные документы
- Тест‑кейсы: [`../tests/test-cases.md`](../tests/test-cases.md)
- Протокол прогона: [`../tests/test-run-2026-01-03.md`](../tests/test-run-2026-01-03.md)

## Условные обозначения
- `TC-XX` — идентификатор тест‑кейса из документа `test-cases.md`.
- Файл скриншота рекомендуется называть **строго** `TC-XX.png` (или `TC-XX-<short-name>.png`), чтобы было удобно ссылаться из тест‑кейсов и протокола.
