Вопросы и ответы

Часто задаваемые вопросы о работе портала

Родителям

Перейдите в раздел «Обратная связь» на главной странице портала. Заполните форму: укажите имя, контактный email, тему и текст обращения. При необходимости приложите файл (до 5 МБ). После отправки вам будет присвоен номер обращения в формате FB-ГГГГ-XXXXXX — сохраните его для отслеживания.

Нет. Форма обратной связи доступна без регистрации и авторизации. Для переписки с конкретным сотрудником и просмотра истории обращений потребуется учётная запись — обратитесь к администратору колледжа для её создания.

Статус обращения обновляется сотрудником колледжа в системе. Если у вас есть учётная запись — войдите в личный кабинет и перейдите в раздел «Сообщения». Без учётной записи уточните статус по номеру обращения, связавшись с колледжем по контактам, указанным в разделе «Контакты».

Принимаются файлы размером до 5 МБ: документы (PDF, DOC, DOCX), изображения (JPG, PNG), таблицы (XLS, XLSX). Файл прикладывается при заполнении формы обратной связи и сохраняется в базе данных системы.
Сотрудникам

Перейдите по ссылке «Войти» в верхнем меню портала. Введите логин и пароль, выданные администратором. При первом входе рекомендуется сменить пароль в настройках профиля.

В личном кабинете перейдите в раздел «Обращения родителей». Выберите обращение из списка — откроется карточка с подробной информацией. Прочитайте текст, при необходимости скачайте вложение. Измените статус обращения (Новое → В работе → Закрыто) и при необходимости свяжитесь с родителем через раздел «Сообщения».

Перейдите в раздел «Сообщения» личного кабинета. В левой панели выберите контакт из списка зарегистрированных родителей. Введите текст в поле внизу и нажмите «Отправить». История переписки сохраняется автоматически.
Технические вопросы

Обратитесь к администратору системы для сброса пароля. Контакты администратора указаны в разделе «Контакты». Самостоятельный сброс пароля по email в текущей версии системы не предусмотрен.

Портал корректно работает в актуальных версиях браузеров: Google Chrome, Mozilla Firefox, Microsoft Edge, Safari. Рекомендуется использовать последние версии браузеров. Поддержка Internet Explorer не предусмотрена.

Обработка персональных данных осуществляется в соответствии с Политикой ЧОУВО «МУ им. С.Ю. Витте» в отношении обработки ПДн и требованиями Федерального закона № 152-ФЗ «О персональных данных». Пароли хранятся в зашифрованном виде. Передача данных осуществляется по защищённому протоколу.
