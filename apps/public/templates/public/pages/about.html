О проекте

Веб-приложение взаимодействия сотрудников колледжа с родителями обучающихся

Цель системы

Портал разработан в рамках выпускной квалификационной работы студента направления «Бизнес-информатика» ЧОУВО «МУ им. С.Ю. Витте». Система автоматизирует процесс взаимодействия сотрудников колледжа с родителями (законными представителями) обучающихся, заменяя разрозненные каналы (телефон, мессенджеры, личные встречи) единой цифровой платформой.

Проблема

До внедрения системы взаимодействие осуществлялось преимущественно по телефону и через мессенджеры, что приводило к потере обращений, отсутствию истории переписки и невозможности контроля сроков ответа. Отсутствовал единый реестр обращений родителей.

Решение

Разработан портал на базе Django 5 и PostgreSQL с ролевой моделью доступа. Система обеспечивает регистрацию обращений, обмен сообщениями, формирование отчётности и публикацию новостей колледжа в одном интерфейсе.

Технологический стек
  • Python 3.12 / Django 5.1
  • PostgreSQL 16
  • Bootstrap 5.3
  • JavaScript (Fetch API)
  • Git / GitHub
